MediRapid

AI-Powered Emergency Medical Services

Features

AI Diagnostics

Advanced symptom analysis and emergency guidance powered by machine learning.

Hospital Locator

GPS-based feature to locate the nearest hospitals quickly.

First Aid Guide

Interactive step-by-step instructions for handling emergency medical situations.

Technologies Used

MediRapid is built using modern web technologies:

  • HTML for structuring the web interface.
  • CSS (Tailwind CSS) for responsive and modern design.
  • JavaScript for dynamic functionality including animations and API integration.
  • Google Maps API for hospital location mapping and navigation.

Installation

  1. 1. Clone the repository: 
    git clone https://github.com/Ankit-CSE-01/MediRapid.git
  2. 2. Navigate to the project directory: 
    cd MediRapid
  3. 3. Set up a local server using tools like XAMPP or WAMP.
  4. 4. Run the application by navigating to http://localhost/MediRapid in your browser.

Usage

  • Find Nearest Hospitals: Click the "Find Hospitals" button to locate nearby hospitals using GPS.
  • AI Emergency Guide: Access the AI-powered chatbot for medical guidance based on symptoms.
  • First Aid Instructions: Get interactive guides for handling common medical emergencies.
  • Provide Feedback: Submit your feedback to help improve MediRapid services.
